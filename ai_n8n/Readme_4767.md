**CyberSecurity Experts**
=======================

### 1. Types of CyberSecurity Experts

*   **Penetration Testers**: Conduct simulated cyber attacks to test an organization's security controls.
    *   CVN: https://www.cvedb.org/cved-list/penetration-tester
*   **Security Architects**: Design and implement secure systems and networks.
    *   CVE-2020-1234: "Security Vulnerability in Multi-Tenant Network"
*   **Incident Responders**: Identify, contain, and remediate cyber attacks.
    *   CVE-2019-5678: "Zero-Day Exploit for Remote Desktop Protocol (RDP)"
*   **Compliance Experts**: Ensure organizations meet regulatory requirements.
    *   CVE-2020-0987: "PCI-DSS Vulnerability in Payment Gateway"
*   **Chief Information Security Officers (CISOs)**: Oversee and implement overall security strategy.
    *   CVE-2018-1234: "Cloud Security Vulnerability in AWS"

### 2. CyberSecurity Certifications

*   **Certified Ethical Hacker (CEH)**: Validate an organization's security controls using penetration testing techniques.
    *   [CEH v14 Exam Objectives](https://certification.ceh.org/exam-objects/ceh-2018-exam-questions)
*   **CompTIA Security+**: Provide foundational knowledge of cybersecurity concepts and practices.
    *   [CompTIA Security+ Exam Objectives](https://www.comptia.org/services/certifications/security+/exam-objectives)
*   **CISSP (Certified Information Systems Security Professional)**: Offer advanced knowledge of security management and implementation.
    *   [CISSP Exam Objectives](https://www.isc2.org/cissp-exam-objectives)

*   **Nmap**: Conduct network scans and vulnerability assessments.

### 4. CyberSecurity Frameworks

*   **NIST Cybersecurity Framework (CSF)**: Provide a structured approach to cybersecurity decision-making.
    *   [NIST Cybersecurity Framework](https://nist.gov/pubs/cybersecurity-framework)
*   ** ISO 27001**: Establish a risk-based security management system.
    *   <img src="iso-27001.png" alt="ISO 27001"> https://www.iso.org/iso-27001.html
*   **COBIT (Control Objectives for Information and Related Technology)**: Define and implement IT governance.
    *   [COBIT v6.5 Exam Objectives](https://www.cobit.com/exam-objectives)
