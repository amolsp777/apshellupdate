Joomla sites running extensions with the Novarain/Tassos Framework face serious risks from newly disclosed vulnerabilities. Independent researcher p1r0x, via SSD Secure Disclosure, uncovered flaws that allow SQL injection and unauthenticated file reads. Attackers can chain these issues for administrator takeover and remote code execution on unpatched systems
